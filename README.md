# Link Project

Este es un proyecto inspirado en **LinkTree** para darme a conocer, exponer mis repositorios y que se puedan contactar conmigo directamente.

## View Project

Para ver el proyecto funcionado visita el repositorio de GitLab: [Link](https://janp-c.github.io/link) y si quieres reutilizarlo en caso de que te guste, te dejo los pasos.

## Usage

```bash
cd existing_repo
git clone https://github.com/janp-c/link.git
```
